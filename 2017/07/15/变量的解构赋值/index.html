简介

  1. 基本用法： let [a, b, c] = [1, 2, 3]
  2. 默认值：let [ n = 1] = []
  3. 解构不成功，变量的值等于undefined

对象的解构赋值

  1. 基本用法：let {foo, bar} = {foo: 'aaa', bar: 'bbb'}
