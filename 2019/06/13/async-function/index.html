Callback， Promise, Async的使用区别

2019-06-13 /
  • ES6
 / 点击 / 阅读耗时 2 分钟

callback

  • 普通情况下，遇到异步情况，可以使用callback的方式，在拿到数据后执行回调函数

Promise

  • 回调方法太多会造成很多问题，所以我们会使用.then()的方式
  • 那么就需要Promise容器的支持

Async

  • 把异步的写法改成同步的写法，就需要使用async
  • async一旦遇到await就会返回，返回的是Promise对象

  • return 返回的值就是then中的参数
